This project uses a datadase collected from vetrinary offices surrounding horse colic, which is a disease horses can get that has to do with their abdomin getting twisted.
This can be fatal to the horse if not treated properly. Not only that it can be hard for the owner to make the right call whether to put to horse down or not. 
The goal of this project was to provide both data to try to inform horse owners of what can lead to higher fatality and to create an AI model to predict which
of the 3 catigories the outcome would be Survive, Dead, or Euthinized.
